# 個人房貸試算器

##  專案簡介 
一個能計算房貸的超強工具！

##  功能特色

* 一鍵得知您的貸款金額，非常方便
* 計算後可以直接前往保險公司網站預約諮詢
* 背景變色提示資料的錯誤或缺漏
* 有趣的gif圖示

##  使用方式
輸入房屋總價、自備款比例、貸款利率、貸款年限、寬限期(選瑱)，就可以算出房貸資訊了。
